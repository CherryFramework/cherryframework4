cherry_register_sidebar()

cherry_register_sidebar(array $args) : string

Wrapper function for WordPress' register_sidebar() function. This function exists so that theme authors can more quickly register sidebars with an HTML5 structure instead of having to write the same code over and over. Theme authors are also expected to pass in the ID, name, and description of the sidebar.

This function can handle the rest at that point.
