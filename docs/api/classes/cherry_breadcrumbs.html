add_path_parents()

add_path_parents(string  $path)

Get parent posts by path. Currently, this method only supports getting parents of the 'page' post type. The goal of this function is to create a clear path back to home given what would normally be a "ghost" directory. If any page matches the given path, it'll be added.

Parameters

string $path

The path (slug) to search for posts by.

map_rewrite_tags()

map_rewrite_tags(integer  $post_id, string  $path)

Turns %tag% from permalink structures into usable links for the breadcrumb trail.

This feels kind of hackish for now because we're checking for specific %tag% examples and only doing it for the 'post' post type. In the future, maybe it'll handle a wider variety of possibilities, especially for custom post types.

Parameters

integer $post_id

ID of the post whose parents we want.
string $path

Path of a potential parent page.
